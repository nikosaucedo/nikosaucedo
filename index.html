What I Learned in This Class

I learned a lot in this class and it mainly helped with touching up on HTML and CSS that I had some knowledge of but also with things that I didn't know about. I learned about flexbox which was an interesting an easy way to organize and contain items on your page. Along with that, learning about W3.CSS, which I had no idea about, was a very easy and in depth way of adding styles, themes, and more advanced changes to your site with minimal effort and coding, which I enjoyed.

Before this class, I had minimal knowledge of WordPress and thought it was very confusing and struggled a lot with it. I think this class helped introduce me to WordPress simply and understandably, which I enjoyed very much because the career I'm hoping to go into will be using WordPress quite a bit along with having the general knowledge of HTML and CSS which is always useful in the field I'm looking to go into.
